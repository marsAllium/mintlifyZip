## Databricks

### Delta Sharing

Allium utilizes [Databricks Delta Sharing](https://docs.databricks.com/data-sharing/index.html) allowing users to access our data in your own Databricks Environment.

### Data Freshness

#### Batch Data

Data freshness on Databricks is typically within 2-3 hours.

#### Real-time Data

Databricks also supports streaming capabilities. Allowing it to achieve sub-minute latency for real-time data:
