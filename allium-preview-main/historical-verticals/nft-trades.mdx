### Sample Query

NFT Volume Across Blockchains.

Copy

```
select
    date(block_timestamp) as date,
    chain,
    sum(usd_amount) as usd_volume
from crosschain.nfts.trades
where block_timestamp >= current_timestamp - interval '90 days'
group by all
```
