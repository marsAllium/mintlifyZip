---
title: "TVL"
---

The schema `defillama.tvl`contains these tables:

| Table                                  | Description                                           |
| -------------------------------------- | ----------------------------------------------------- |
| historical\_agg\_token\_amounts        | Daily historical token amounts per protocol           |
| historical\_agg\_token\_values         | Daily historical token values per protocol            |
| historical\_agg\_tvl                   | Daily historical TVL per protocol                     |
| historical\_token\_amounts\_per\_chain | Daily historical token amounts per protocol per chain |
| historical\_token\_values\_per\_chain  | Daily historical token values per protocol per chain  |
| historical\_tvl\_per\_chain            | Daily historical TVL per protocol per chain           |
