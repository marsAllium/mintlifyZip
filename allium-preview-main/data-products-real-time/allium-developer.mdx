Allium App: Developer Interface

### Data Freshness

***Allium Developer*** offers fast query speed (50-100ms) and **real-time data freshness (3-4s)** at **engineers**' fingertips.

### Query Engine

Allium Developer runs on [ClickHouse](https://clickhouse.com/) for Solana and [Postgres](https://www.postgresql.org/) for all other chains.

### Schema Catalog

Allium Developer provides real-time access to all blockchains listed on this [page](/realtime/supported-blockchains).
