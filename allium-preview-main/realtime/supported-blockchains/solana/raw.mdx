---
title: "Raw"
---

Our Solana raw schema contains the following tables:

| Schema | Description          | Tables                                                                |
| ------ | -------------------- | --------------------------------------------------------------------- |
| Raw    | Raw blockchain data. | blocks, transactions, instructions, inner\_instructions, pubkey\_maps |

Our raw solana data contains the last 7 days of history. Additional support for historical Solana data is in real time via request or available through the explorer interface.
