Decoded ERC token metadata.

PSQLAPI

Query ERC tokens by contract address.

Copy

```
select * from base.erc20_tokens 
where address = '0x833589fcd6edb6e08f4c7c32d4f71b54bda02913' -- USDC Contract
```
