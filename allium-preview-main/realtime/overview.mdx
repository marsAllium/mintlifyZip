There are 2 ways to access Allium's realtime data (\<3-5 seconds) today:

1. **Pull:** Allium has APIs that allow customers to pull data from our realtime Postgres database. This can be either [prebuilt endpoints](https://docs.allium.so/products/allium-developer/wallet-apis) (Wallet Activities, Balances, Prices) or [custom raw SQL](https://docs.allium.so/products/allium-developer/sql-api).

2. **Push:** Allium can push data in realtime through event buses like Kafka, PubSub and SNS.
